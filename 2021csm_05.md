## 새로 배운 내용

1. RAID란 여러개의 디스크를 하나로 사용하는 것이다.
2. 소프트웨어 RAID의 종류
  * LinearRAID : 순서대로 사용
  * RAID0 : 모든 디스크에 동시에 저장
  * RAID1 : 하나의 데이터를 두 개의 디스크에 저장
  * RAID5 : RAID0과 RAID1의 기능을 모두 수행 (패리티 사용)
3. RAID 과정
  * 파티션 상태 확인
  * RAID 생성
  * 포맷 (파일시스템 생성)
  * 마운트할 디렉터리 생성
  * 마운트
  * 부팅 시 자동으로 마운트 설정
  * fdisk와 mdadm 명령어가 쓰인다. 까먹으면 tldr을 사용하자.
