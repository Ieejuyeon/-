# 새로배운 내용

## 디스크 관리

~~~
* SATA : 하드디스크 또는 광디스크의 연결에 사용
* SCSI : 주변기기 연결에서 사용
* 파티션 : Primary, Extended, Logical로 나뉘며 하드디스크를 나눠쓰는 것
* RAID : 파티션을 할당한 하드디스크 여러개를 하나처럼 사용하는 방식
~~~

---

### 하드디스크 추가하기

1. vmware 에서 하드디스크 1개 추가
- SCSI / 1GB
2. 파티션 할당
- fdisk /dev/sdb (추가한 하드디스크웨어, 파티션은 sdb1로 생성된다)
- n,p,1,enter,enter,p,w
3. 파일시스템 생성
- mkfs.ext4 /dev/sdb1
4. 마운트
- mkdir /mydata
- mount /dev/sdb1 /mydata
5. 부팅시 자동 마운트 설정
- vi /etc/fstab
- (vi 내 작성) /dev/sdb1 /mydata ext4 defualts 0 0
- reboot

---

### 레이드 구성하기

1. 하드디스크 파티션
- 위의 내용대로 하면 된다.

2. mdadm 설치
- apt install mdadm

3. 레이드 생성
- mdadm --create /dev/md9 --level=(레이드레벨) --raid-devices=(레이드개수)  /dev/(파티션1) /dev/(파티션2)

4. 파일시스템 생성
- mkfs.ext4 /dev/md9

5. 마운트
- mount /dev/md9 /(마운트할 디렉터리이름)
- mount /dev/md9 /(마운트할 디텍터리이름)

6. 부팅시 자동 마운트 설정
- vi /etc/fstab
- /dev/md9 /(마운트된 디텍터리이름) ext4 defaults 0 0

---

### 확인 하기

* 파티션 : fdisk -l /dev/sd*
* 레이드 : mdadm --detail --scan
