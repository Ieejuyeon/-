## 새로 배운 내용

### Shell
1. Shell이란 : 사용자와 커널사이 명령을 전달해줌
2. Shell Script : 명령어 집합
### Shell Script
1. 환경변수 print env로 확인
2. 작성방법
  * touch ?.sh
  * vi ?.sh
  * bash ?.sh
3. 변수설정
  * 값이 입력되면 자동설정됨
  * 숫자는 expr
  * 수식은 `?`
  * 변수를 사용할때는 앞에 $붙이기
  * 파라미터 변수 $n : n번째 항
4. 조건문
  * if [] then elif else fi
  * case 변수 in
    조건 | 조건 | 조건 )
      결과
    조건 | 조건 | 조건)
      결과
    *)
      결과
    esac
 5. 반복문
   * for (i=0  i<n i++)
   * for 변수 in 범위; do
      결과
      done
   * while [조건]; do
      결과
      done
 6. 사용자 정의함수
   * 함수이름 () {실행내용}
   * 파라미터로 호출가능
   * 지역, 광역으로 나뉘어 사용가능
 7. 기타
   * eval : 문자열을 명령어로 해석
   * set : 결과를 파라미터로 지정 가능
 8. 작업예약
   * date, at, cron 이용
