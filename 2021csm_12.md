## 새로배운내용

### Mail Server

* SMTP : 클라이언트가 메일을 보낼때, 메일 서버끼리 메일을 주고 받을 때
* IMAP, POP3 : 메일 서버에서 클라이언트로 메일을 가져올 때
* IMAP : 모든 장치에서 가능, 메일을 복사한다. 서버동기화로 실시간확인가능.
* POP3 : 로컬 장치로만 가능, 메일을 가져온다. 서버의 저장공간을 차지하지않음.

### Sendmail
* MTA : Mail Transfer Agent
* 메일전송을 위한 패키지

<br>

* MUA : Mail User Agent (클라이언트)
* MDA : Mail Delivery Agent
* MRA : Mail Retrieval Agent

### 메일 전송 과정

* MUA - MTA - MDA - Internet - MTA - MDA - 메일박스 - MRA(Dovecot) - MUA

### 실습과정

* DNS Server 구축 (bind9, bind9utils)
* Mail Server 구축 (sendmail)
* Evolution을 이용한 메일 전송

<br>

### 회고

* 실습 난이도는 생각보다 쉬웠다. 원격접속이나 DNS Server구축하는 것을 이미 해봤어서 그런 것 같다.
* 그렇지만 네트워크 환경 구성도가 잘 이해되지 않는다. 복습을 해야겠다.
* 문제가 생긴적은 없었다. 다만 유저설정 부분에서 비밀번호를 잊었고, 그것을 어떻게 재설정하는지 배웠는데 까먹었었다.
* 찾아보니 passwd [username] 으로 간단히 해결할 수 있었다.
