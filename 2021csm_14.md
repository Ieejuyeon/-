# 새로배운내용

## 방화벽
* 1세대 방화벽 : 패킷을 하나하나 까서 검사한다.
* 2세대 방화벽 : 세션단위로 검사한다.
* 3세대 방화벽 : 응용프로그램에 미칠 영향을 분석한다.
* IP 마스커레이딩 : IP를 공인 IP로 바꾸어 인터넷에 접속한다.
* 정책을 만들어 관리한다.

## iptables
### chain
* PREROUTING : 장치에 패킷이 도착했을때
* POSTROUTING : 패킷이 장치밖으로 나갈때
* INPUT : 호스트 컴퓨터로 들어오는 모든 패킷
* OUTPUT : 외부로 내보내는 패킷
* FORWARD : 지나가는 패킷
### 규칙
* ACCEPT(허용)
* DROP(버림)
* REJECT(버리고응답)
* RETURN(패킷처리계속)
