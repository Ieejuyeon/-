## 새로배운내용

### LAMP : 웹구축환경
* Linux (OS)
* Apache HTTP Server (HTTP 웹 서버)
* MySQL (DB Server)
* PHP (동적 웹 페이지)
* Client -> Web Server -> CGI Language -> DB server

### Web Server
* 클라이언트로부터 HTTP 요청을 받고 웹페이지를 반환한다.
* 이 기능을 수행하는 컴퓨터
* Apache : 오픈소스, 스레드 형태의 프로세스 - 대규모의 요청 처리가 힘들다.
* http 443 : 암호화된 포트
