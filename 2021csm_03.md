## 새로 배운 내용

1. 리눅스 디렉터리 구소
  * /bin : 명령어를 저장함
  * /cdrom : CD-ROM이 마운트 되는 폴더
  * /etc : 설정 파일
  * /lib : 커널모듈파일, 라이브러리
  * /media : 탈부착할 수 있는 장치의 마운트포인트로 사용됨
  * /proc : 가상파일시스템 - 디스크에 존재하지 않고 메모리에 존재하는 작업을 저장함
  * /root : 개인 디렉터리
  * /sbin : 시스템 관리자용 명령어
  * /usr : 일반 사용자용 명령어

<br>

2. 사용자 관리
~~~
  * 리눅스 서버 1대에 여러 명의 사용자가 동시 접속 가능
    - 순차적이지만 매우 빠른 속도로 처리가 되므로 동시접속이라고 볼 수 있다.
  * 슈퍼 유저 (Root)는 일반 사용자를 만들고 임의의 그룹에 넣을 수 있다.
~~~  
~~~
  * 사용자 정보 확인 : /etc/passwd
  * 그룹 정보 확인 : /etc/group
  * /etc 앞에 /less 를 붙이면 요약, /tail을 붙이면 마지막 10줄
~~~
~~~
  * 기본폴더 생성 : cd/etc/skel
  * 그룹생성 : groupadd <그룹명>
  * 계정생성 : adduser <사용자명>
  * 기본그룹 변경 : usermod --gid <그룹명> <사용자명>
  * 보조그룹 변경 : usermod --groups <그룹명> <사용자명>
  * 사용자 삭제 : userdel <사용자명>
  * 사용자 비밀번호 변경 : passwd <사용자명>
~~~

<br>

3. 파일속성
* 소유권 : 생성 및 관리
* 허가권 : 접근 및 실행
* 소유권 변경
  - chown <사용자명> <파일명>
  - chgrp <그룹명> <파일명>
 * 허가권 변경
  - read, write, execute 각각의 권한을 2진수로 표기
  - 10진수로 변환하여 하나의 숫자로 나타냄
  - 소유자, 그룹, 그 외 사용자 순으로 위의 세 숫자를 나열한 것을 (예시) 554라고 할때
  - chmod 554 <파일명> 으로 적용함
