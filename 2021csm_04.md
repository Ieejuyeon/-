## 새로 배운 내용

### 프로그램 설치 방법과 과정 

* 터미널 이용

~~~
1. 패키지 저장소 설정
    - sources.list 파일안에 들어있다.
    - sources.list.d는 디렉터리로, 부수적인 저장소를 보관하는 곳이다.
  
2. 프로그램 설치
    - apt list (프로그램명) : 설치 및 업데이트 가능여부 확인
    - apt install (프로그램명) : 설치

3. 프로그램 삭제
    - apt remove (프로그램명) : 해당 패키지만 삭제
    - apt autoremove (프로그램명) : 관련된 패키지와 설정파일 모두 삭제
~~~

* Ubuntu Software 이용
* 해당 웹사이트에서 직접 다운로드
* 웹사이트에서 가장 최신의 버전을 다운받을 수 있음

## 문제 발생과 해결과정

~~~

   엔터를 쳤는데 프로그램이 안돌아가는건줄 알고 몇번이고 재부팅 하였다. (...)
