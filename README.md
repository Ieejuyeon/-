## 새로배운내용

<br>

**1. 종료 명령어**
* Poweroff
* Shutdown -P now
* Halt -p
* Init 0
* 관리자만 종료하는경우 로그아웃을 해야함

**2. 그 외 명령어**
* pwd : 현재위치 보기
* cd (파일이름) : 파일위치로 이동
* cp (파일1)(파일2) : 파일1을 파일2로 복사
* mkdir 폴더생성
* Touch (파일이름).txt : txt파일생성
* Man (명령어) : 명령어 도움말
* Cat (파일이름).txt : 파일 출력 

**3. 리디렉션**
* man (명령어) > (파일이름).txt
  출력을 파일로 저장
* man (명령어) >> (파일이름).txt
  출력도 하고 파일로 저장도함
* sort < (파일이름).txt
  abc 순 정렬
 
**4. 압축**
* Tar
  Tar cvf (압축할이름).tar (압축할 폴더)
* Gzip
  Tar zcvf (압축할이름).gz (압축할 폴더)
* Bzip2
  tar jcvf (압축할이름).bz2 (압축할 폴더)
